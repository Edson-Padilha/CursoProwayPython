Comando deverá ser executado no terminal todo inicio de projeto: 
django-admin startproject meu projeto
Entrar no diretório onde está os arquivos e executar no terminal:

python manage.py
Vai listar alguns comandos 

Subindo o servidor na porta localhost:8000
python mange.py runserver

Para alterar a linguagem para português:
Vai em settings: 

LANGUAGE_CODE e altera para pt-br
TIME_ZONE = 'America/Sao_Paulo'

Comando para atualizar página
CTRL + SHIFT + R
Desta forma não pega o armazenamento no cache

Comando para atualizar o banco de dados:
python manage.py migrate

Comando para criar super usuário:
python manage.py createsuperuser 

Comando para criar App:

python manage.py startapp meua App
